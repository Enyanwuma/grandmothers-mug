Grandmother's Ceramic Mug

blender4.x compatible licenseMIT tutorialbeginner → intermediate made withpatience

"It wasn't just a model. It was my history, recreated by my hands, using a tool that cost nothing but patience."

An interactive, story-driven web guide for modeling a vintage 1970s ceramic coffee mug in Blender — complete with a chipped rim, hand-painted faded flowers, ceramic glaze material, and honest, imperfect geometry. Built as a single-file HTML experience with no dependencies.

✦ About This Project

This tutorial was born from a real Saturday night — someone alone, with no plans, who decided to model their grandmother's old ceramic mug from memory. No tutorial. Just observation. This guide recreates that process as a step-by-step walkthrough, but keeps the spirit of that original spontaneous act of creation.

The guide covers everything from workspace setup to final Cycles render, with each step grounded in what makes this specific mug feel real: the tapering silhouette, the weight of the handle, the chip's tiny shadow.

✦ Phases

Phase Title Steps Key Skills
01 Setup 3 steps Scene cleanup, reference images, units
02 The Body 5 steps Cylinder, loop cuts, rim, foot ring
03 The Handle 3 steps Bezier curves, bevel, Bridge Edge Loops
04 Soul 2 steps Proportional editing, Subdivision Surface, Crease
05 Surface 3 steps Principled BSDF, UV unwrap, Texture Paint, noise
06 The Scene 3 steps HDRI lighting, area light, camera composition

✦ Getting Started

Prerequisites

  • Blender 4.0+ (free — blender.org)
  • A modern web browser to follow the interactive guide
  • 2–4 hours of quiet time. A Saturday night works well.

Open the Guide

bash# Clone the repository
git clone https://github.com/blender-artists/grandmothers-mug.git

# Open the guide in your browser
open grandmothers-mug/index.html

# That's it. No npm install. No build step. Just a mug.
🌿
TipKeep the guide open on one monitor and Blender on the other. Each step is designed to be followed in real time.

✦ Key Shortcuts Referenced

The guide surfaces Blender shortcuts contextually. Here's a quick reference for the most-used ones:

  • Tab — Toggle Edit / Object mode
  • Ctrl + R — Loop Cut
  • Alt + Click — Select edge/face loop
  • O — Toggle Proportional Editing
  • Shift + E — Crease selected edges
  • U — UV Unwrap menu
  • F12 — Render
