A Blender Tutorial

Grandmother's Ceramic Mug

From a default cube to a memory. A step-by-step journey through modeling, shaping, and rendering a weathered ceramic mug — imperfections and all.

↓ Scroll to begin

Phase 01 — Setup

Preparing the Workspace

Before you sculpt a memory, you need a clean stage. A few minutes here saves hours of confusion.

1

Open Blender & Clear the Scene

Launch Blender. You'll be greeted by the startup splash screen and the default scene — a cube, a camera, and a light. Select all objects and delete them. We start fresh.

ASelect All X→ Delete
2

Add a Reference Image (Optional but Powerful)

Find an old photo of a ceramic mug — or sketch the rough shape on paper. In the front viewport, add an image empty as a backdrop reference. This anchors you to reality.

ShiftAAdd → Image → Reference
The Story TipStudy the curve of the handle. The thickness of the rim. The way light catches a chip. Observation before execution.
3

Set Up Your Units & Overlay

Go to Scene Properties → Units → set Metric, scale 0.001 (millimeter). Turn on wireframe overlay so you can see through your mesh as you work. Small details live in the edges.

AltZToggle X-Ray view
✦ ✦ ✦
Phase 02 — The Body

Shaping the Mug from a Cylinder

"I started with a cylinder." The same as you. The same as everyone.

4

Add a Cylinder

Add a cylinder with 32 vertices. This is your mug's skeleton. Keep the depth close to your mug's real proportions. Delete the top and bottom faces — we'll rebuild them with more control.

ShiftAMesh → Cylinder
Settings32 vertices · Cap Fill Type: Nothing · Depth: 0.1m (scale later)
5

Enter Edit Mode & Loop Cut the Body

Enter Edit Mode. Add 4–5 horizontal loop cuts along the body. This gives you rings of vertices to push and pull — creating the subtle tapering of a real mug. The middle often widens slightly; the base narrows.

TabEdit Mode CtrlRLoop Cut → scroll to add 4 cuts
6

Scale Rings to Shape the Silhouette

Select each edge loop and scale it in the X-Y plane to shape the mug's profile. Select a loop by Alt-clicking it, then scale. A real mug isn't a perfect cylinder — it breathes.

Alt+ClickSelect Loop SShiftZScale on X/Y only
Feel itBottom loops: slightly smaller. Mid-body: widest. Just below rim: narrows again. Trust your eye.
7

Cap & Extrude the Rim

Select the top edge loop. Extrude it inward to form the rim's thickness — old ceramic mugs have weight, presence. Then extrude down again to form the inner wall, and cap the bottom inside.

EExtrude SScale (inward) FFill (close bottom)
8

Add & Shape the Bottom

Select the bottom edge loop. Extrude inward slightly to create a foot ring (that tiny ridge under real mugs). Fill the center. This is the detail no one notices until it's missing.

AltClickBottom loop ESExtrude → Scale in
✦ ✦ ✦
Phase 03 — The Handle

The Curve of the Handle

The handle is where the mug becomes personal. It's how your grandmother held it. Get the curve right.

9

Create a Bezier Curve for the Handle Path

Add a Bezier curve. This will be the spine of the handle. Shape it in the side view to follow the oval arc of a typical mug handle — top attachment point, bottom attachment point, with a comfortable outward bow.

ShiftACurve → Bezier Numpad 3Side view
10

Add a Circular Bevel Profile

In the curve's properties, set the Bevel Mode to "Round" with a small depth. This gives the handle a cylindrical cross-section. The handle feels solid now, not just a line through space.

SettingsCurve Properties → Geometry → Bevel → Round · Depth: ~0.008m
11

Convert to Mesh & Attach

Convert the curve to a mesh. Then manually select the top and bottom face loops of the handle and the corresponding spots on the mug body, and use Bridge Edge Loops to create seamless attachment — like the handle was always there.

AltCConvert to Mesh CtrlJJoin with mug body
BridgeSelect two edge loops → Right Click → Bridge Edge Loops. Adjust cuts to 3–4 for smooth blending.
✦ ✦ ✦
Phase 04 — Soul

The Chip, the Imperfections

"The chip's tiny shadow. The imperfect rim." This is what separates a model from a memory.

12

Model the Chip on the Rim

Select a small cluster of vertices along the rim edge. Use Proportional Editing with a tiny radius. Pull them slightly downward and inward to create that worn chip — the one that's been there so long it became part of the mug's character.

OToggle Proportional Edit GGrab → move down slightly ScrollAdjust influence radius
13

Add a Subdivision Surface Modifier

Add a Subdivision Surface modifier (level 2 viewport, 3 render). This smooths everything beautifully while preserving your hand-shaped proportions. The mug goes from blocky to ceramic-soft.

Properties Panel→ Wrench → Subdivision Surface
Control sharpnessSelect edge loops near the rim and base, then Crease them (Shift+E → 1) to keep them crisp while the rest stays smooth.
✦ ✦ ✦
Phase 05 — Surface

Faded Flowers & Ceramic Glaze

The 70s ceramic glaze. The hand-painted flowers, now soft and sun-bleached. Material is memory made visual.

14

Create the Ceramic Base Material

In the Material Properties, create a new material. Use the Principled BSDF shader. Set Base Color to a warm off-white (creamy ceramic), Roughness to ~0.25 (slightly glossy glaze), and Specular to ~0.6.

Shader Editor→ Principled BSDF
ColorHex #E8C99A or similar warm bisque. Not pure white — aged ceramic absorbs warmth.
15

Paint the Flowers with Texture Paint

UV Unwrap the mug. Switch to Texture Paint mode. Create a new texture image. Paint simplified floral shapes in muted rose pink and faded blue. Keep them imprecise — they were hand-painted in 1972, not computer generated.

UUV Unwrap (Smart Project) Texture Paint Mode
FadingReduce brush strength to ~0.3 and paint multiple strokes to build up soft, transparent color — simulating decades of dishwasher wear.
16

Add Subtle Surface Noise for Age

In the Shader Editor, add a Noise Texture node into the Roughness channel. Set scale very high (~200), strength very low (~0.05). This gives microscopic variation — the mug looks like it's been through 50 years of mornings.

Node chainNoise Texture → ColorRamp (0.1 to 0.35 range) → Principled BSDF Roughness
✦ ✦ ✦
Phase 06 — The Scene

Placing the Mug in Light

"Sitting on a virtual table." The context completes the story.

17

Add a Simple Table Surface

Add a plane, scale it large. Assign a wood or cloth material — something warm, textured. Use an HDRI environment for ambient light. A single area light from the upper left creates that quiet Saturday night lamp glow.

ShiftAMesh → Plane
18

Set Up Lighting

Use a warm area light (color ~3200K) above-left at medium power. Add a soft fill light from the right at very low power. This isn't a studio — it's a kitchen counter at 11pm. The light should feel honest.

HDRIWorld Properties → Use Nodes → Environment Texture → Load a warm indoor HDRI (Poly Haven has free ones). Rotate until the light direction feels right.
19

Compose the Camera Shot

Position the camera slightly above eye level, tilted about 15° down. Focal length 85mm feels like a portrait lens — intimate, honest. Frame so the handle faces the viewer and the flower motif is visible. The chip should catch the light.

Numpad 0Camera view GMove camera

When I finally hit render and saw that digital mug — imperfections and all — emotion hit me.

Final Render Checklist

Cycles render engine (not EEVEE) for realistic shadows
Samples: 512+ (lower for preview, higher for final)
Denoiser: ON (Intel Open Image Denoise)
Color Management: Filmic · Contrast: Medium High
Output: PNG · 1920×1080 or 2:3 portrait for impact
The chip is visible · The flowers are soft · The glaze catches light
